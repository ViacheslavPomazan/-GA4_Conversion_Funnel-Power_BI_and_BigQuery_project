# -GA4_Conversion_Funnel-Power_BI_and_BigQuery_project
An end-to-end e-commerce analysis project that uses SQL (BigQuery) to process 4M+ rows of raw GA4 data and Power BI to visualize conversion funnel performance across various user segments.

## Goal
The primary goal is to visualize the user journey from the 'session_start' to the final 'purchase' through such stages as 'view_item', 'add_to_cart', 'begin_checkout', 'add_shipping_info', 'add_payment_info', identifying where potential customers drop off and how different user segments behave throughout the funnel.
